## About me

- Platform Engineer shipping a **multi-tenant GCP landing zone** for a bank, built on [Cloud Foundation Fabric / FAST](https://github.com/GoogleCloudPlatform/cloud-foundation-fabric).
- Currently running the **AI / RAG platform** side: private GKE, Cloud SQL, Airflow 3, Langfuse, FastAPI + Next.js apps, all on top of Workload Identity Federation.
- CS student at **UTN-FRSR** (4th year), based in **Mendoza, Argentina**.
- I write about the production infra problems that **no tutorial warns you about**: Helm hook ordering, WIF multi-branch CDs, silent Airflow rollouts, Cloud SQL Proxy in migration jobs.

Those notes live here: [**war-stories**](https://github.com/agustin2505/war-stories)
